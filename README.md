# SQL_music_store_analysis

## Project Overview
This project involves analyzing the Chinook Music Store database using SQL.
I solved business-oriented questions to understand customer behavior, sales trends, artist performance, and genre popularity.

## Tools Used
- SQL
- MySQL Workbench

## What I Did
- Used JOINs to combine multiple tables
- Applied GROUP BY and aggregate functions (SUM, COUNT, AVG)
- Wrote subqueries and CTEs
- Identified top customers and high-revenue cities
- Analyzed customer spending patterns
- Found most popular genres by country
- Determined artist-wise revenue contribution

## Skills Demonstrated
- SQL Query Writing
- Data Aggregation
- Business Analysis
- Relational Database Understanding
